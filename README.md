# Expert TextingThis is a repo for managing machine readable artifacts for the [Expert Texting APIs](http://www.experttexting.com/features/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Expert Texting APIs](http://www.experttexting.com/features/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.experttexting.com/features/)- [Developer](http://www.experttexting.com/features/)- [Twitter](https://twitter.com/Experttexting)The [APIs.json](https://github.com/api-evangelist/expert-texting/blob/master/apis.json) index provides links to all the relevant links for the Expert Texting APIs, including the OpenAPI and Postman if they are present.
